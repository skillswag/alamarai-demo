üåê Deployment Option 1: Netlify (Recommended)

Why Netlify? Free hosting, automatic SSL, custom domains, and one-click deployment.

Method A: Drag & Drop (Easiest)

Step 1: Prepare Your Files

Make sure your project folder contains all files:

  • index.html
  • css/ folder with styles.css
  • js/ folder with main.js
  • assets/ folder with all images
  • netlify.toml (configuration file)

Step 2: Visit Netlify

Go to https://netlify.com

Click "Sign up" if you don't have an account (it's free!)

Step 3: Deploy

  1. Once logged in, you'll see the deployment page
  2. Drag and drop your entire almarai folder onto the deployment zone
  3. Wait 10-30 seconds for deployment
  4. Your site is live! üéâ
‚úÖ Done! Netlify will give you a URL like: https://random-name-12345.netlify.app
You can customize this URL in your site settings.

Step 2: Connect to Netlify

  1. Log in to Netlify
  2. Click "New site from Git"
  3. Choose "GitHub" and authorize
  4. Select your almarai-demo repository
  5. Leave build settings empty (static site)
  6. Click "Deploy site"
‚úÖ Benefit: Now every time you push to GitHub, Netlify automatically updates your site!

üìä Performance Optimization Tips

Image Optimization

  • Convert SVG placeholders to real JPG/PNG
  • Compress with TinyPNG or ImageOptim
  • Use WebP format for better compression
  • Keep files under 200KB each

Code Optimization

  • Minify CSS and JavaScript
  • Enable gzip compression (automatic on Netlify)
  • Use CDN for assets (automatic on Netlify)
  • Enable caching (configured in netlify.toml)

SEO Optimization

  • Add meta descriptions to HTML
  • Create sitemap.xml
  • Add robots.txt
  • Submit to Google Search Console

Analytics

  • Add Google Analytics
  • Enable Netlify Analytics
  • Track visitor behavior
  • Monitor site performance

üêõ Troubleshooting Common Issues

Images Not Loading

Problem: Broken image links on live site

Solution:

  • Check file paths are relative (not absolute)
  • Verify image files exist in assets/images/
  • Ensure filenames match exactly (case-sensitive)
  • Check file extensions (.jpg, not .JPG)

JavaScript Not Working

Problem: Slider or filtering not working

Solution:

  • Check browser console for errors (F12)
  • Verify main.js is loaded
  • Check script tag in HTML is before
  • Clear browser cache and refresh

Mobile Layout Issues

Problem: Site looks broken on mobile

Solution:

  • Check viewport meta tag in HTML
  • Test with browser DevTools mobile view
  • Verify CSS media queries
  • Check for fixed widths in CSS
